# Legacy Material

This folder preserves the historical origin of the spectral topic
modelling idea used by the project.

It is not production code and not the authoritative implementation, but
it is still scientifically important because it contains:

- the original exploratory notebook that tests three spectral-to-corpus
  mappings
- the paper draft that formalizes the HSI-to-LDA analogy and the
  hierarchical inference idea

## What To Read First

- [wiki/legacy-audit.md](../wiki/legacy-audit.md)
- [notebooks/README.md](notebooks/README.md)
- [papers/README.md](papers/README.md)

## Working Rule

Preserve the material, audit it, and mine it for useful ideas.
Do not confuse it with the reusable pipeline or the validated local-core
workflow.
